### 🚀 Highlighted Architectures

While my **Pinned Repositories** below showcase my featured open-source work, here are a few standout systems I've architected:

*   **[ScoutSearch Engine](https://ali00922-scoutsearch.hf.space/):** A millisecond-latency natural language search engine built from scratch. Utilizes a custom *Sharded Inverted Index*, *O(1) Forward Index*, and *Prefix Tries* to dynamically map queries over 19,000+ multidimensional profiles using TF-IDF.
*   **[EmojiMood CNN](https://emoji-weex.onrender.com):** An edge-optimized facial emotion recognition system. Compressed a standard ML model by 75% into an 8.4MB TF-Lite file, achieving 120ms inference speeds for real-time live web deployment via Docker/Render.
*   **[TaskMate AI](https://takymasky.netlify.app/):** A productivity platform integrating DeepSeek AI for real-time code generation and OpenAI for automated workflows, built on a Firebase/JS architecture.
